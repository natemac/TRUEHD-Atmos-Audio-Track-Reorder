# TRUEHD-Atoms-Audio-Tracker-Reorder
Windows drag-and-drop tools to losslessly reorder MKV audio tracks. Move AC-3 / E-AC-3 to position 1 so Plex direct-plays instead of transcoding TrueHD/Atmos. Single-file interactive mode plus folder batch mode with CSV logging. Pure mkvmerge remux — no re-encoding, with safe .bak backups.
